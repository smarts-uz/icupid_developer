Chat Widget

After chat installation you can find chat-widget demonstration page http://yourdomain/chat71/chat-widget/index.htm  
File structure is as follows:

 

  chat-widget      
    css      
    img      
    js      
      index.htm      

 

Chat widget shows the conversation in one of your chat rooms. It can be used to advertise your new chat system to all the users of your web-site.
