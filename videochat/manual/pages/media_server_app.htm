Media Server Installation

FMS Edition
Copy all the directories from flashcoms/media_server/fms/ to '<fms_root>/applicactions/' directory
 
Wowza Edition
1. Copy the following directories to your Wowza root directory:
flashcoms/media_server/wowza/applications
flashcoms/media_server/wowza/conf
flashcoms/media_server/wowza/lib
2. Restart Wowza media server
 
Red5 Edition

1.Copy all the directories from flashcoms/ media_server/red5/ to '<red5_root>/webapps/' directory

2. Restart Red5 media server. Note, recommended Red5 version is 1.0RC1
