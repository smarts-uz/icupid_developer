Chat Sessions

With the help of "Sessions" table in Admin Panel you are able to monitor all currently active sessions. User1 and User2 fields stand for the names of two opponents.

"Started" field fixes time, when both users opened messenger windows for conversation. Overall time of current session is stored in "Duration" column. You may forcibly close certain session by clicking on "Close" button in "Actions" column.

Note, that only session between two selected users will be shut down. If any of the opponents has another active session with different user - it will not be affected.
