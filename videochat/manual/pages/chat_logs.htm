Chat Logs

You can retrieve all chat conversations by enabling chat logs option in admin panel. To do that go to Admin panel/Logs, enable chat logs option, select how often you want to receive logs and specify handler URL

If enabled chat logs will be sent to your server in the following XML format:

 <chatLogs ts="TIMESTEMPHERE">

 
  <m>
    <r>room1</r>
    <f>29</f>
    <to>72</to>
    <w>true</w>
    <dt>1247580882330</dt>
    <c> Hi marc</c>
  </m>
 
  ...
</chatLogs>

Where:

ts server time stamp
m message node
r room ID
f from user id
to to user id
w is whisper or not (true,false)
dt message sent time stamp
c message content
