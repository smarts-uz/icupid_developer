Error Codes

# 0.
Description Connection to chat server failed.
Troubleshooting Confirm the following:

1. Chat server is up and running.
2. Port 8088 is open on the server.
3. Firewall doesn't block outgoing connections to port 8088 on your local PC.
4. Chat server is running on the right domain (specified in your license).
 
# 1.
Description Chat server rejects connection. Response from chat server was received however server rejected connection to the application.
Troubleshooting Make sure that valid domain name is specified in settings/main.properties. Server refuses connection if flash client connects from domain different from specified in main.properties.

# 2.
Description Connection to media server failed.
Troubleshooting

Confirm the following:

1. Media server is up and running.
2. Port 1935 is open on the server.
3. Firewall doesn't block outgoing connections to port 1935 on your local PC.
4. Media server is running on the right domain (specified in the settings in admin panel).


# 3.
Description Media server rejects connection.
Troubleshooting

Confirm the following:

For FMS: file FMS_root/applications/chat71/main.asc has ACCEPT_DOMAINS variable that contains URL of your web-site.
For Red5: file red5_root/webapps/chat71/WEB-INF/red5-web.properties has variable webapp.acceptDomains that contains URL of your web-site.
For Wowza: file Wowza_root/conf/chat71/Application.xml has property ACCEPT_DOMAINS and it's value is URL of your web-site.
