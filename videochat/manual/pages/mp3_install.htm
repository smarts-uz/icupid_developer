MP3 Player Installation

1. User SQL dump file to create required database table. The file can be found inside of installation package -

www/chat7/mp3player/sql/mp3player.my.sql

2. Copy everything from "WWW" folder of the downloaded archive in to the folder we have created in the previous step. File structure will be as follow:

// DB connection

define("DB_HOST", "localhost");

define("DB_USER", "db_user");

define("DB_PASSWORD", "db_pass");

define("DB_NAME", "mp3player");

define("DB_PREFIX", "chat7_"); // do not change

3. Set writing permissions 777 for the following folder which is used to store MP3 files -

www/chat7/mp3player/mp3player/files/mp3

4.  Enable MP3 player in admin panel (Media player section)
