Users Online

"Users Online" table in Admin Panel shows list of users, currently registered on-line in the system. If user browses any page of your web-site where presence code is embedded, he immediately appears in "Users Online" table.

Click "Ping" button in "Actions" column next to the user name to know, when the last ping was sent to Streaming Media Server, requesting user's on-line status. Click "Disconnect" button in "Actions" column next to the user name to forcibly close the session.
