Friends List customization

Friends List can be customized to match the website page feel and look. You are able to set width, height and background color:

 <script type="text/javascript"> 

       z5messenger.uid = '2010';
       z5messenger.flDefTab = 0;
       z5messenger.flMinimizedByDef = true; 
        z5messenger.ShowFriendList(250, 400, 0xff0000);
</script>

Where:

z5messenger.flDefTab Indicates default tab open upon loading. 0 - My Friends; 1 - Online Users
z5messenger.flMinimizedByDef True - Friends List is minimized. False - Friends List is open.
250 Friends List width in pixels
400 Friends List height in pixels
0xff0000 Background color in HEX format.
