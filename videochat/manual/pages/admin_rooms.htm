Room Management

In order to create new rooms go to "Rooms" section of administration console:

Where:

Room Name The name of the room, which appears on chat home screen.
Room Color The colour of the room, which appears in the select drop down box.
Type Indicates whether the room is a chat based or whiteboard.
Access Specifies whether the room is public or password protected.
Password Password for the Password protected room.
Max Users Determines the maximum number of users allowed for this room (optional).
User Groups Enter user levels separated by comma to specify which are allowed to see the room or leave empty to allow everyone (only if Limitation plugin is enabled).
Description Some text, which describes the room (optional).
