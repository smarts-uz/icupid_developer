Friends List setup

1. Friends List widget is a flash based application to be embedded into any page of your website. With Javascript reference added to the needed page (JS and Presence pixel setup -> step#1) put the following piece of code in to the place you want Friends List to appear.

 <script type="text/javascript"> 

z5messenger.uid = '2010';
z5messenger.ShowFriendList();
</script>

Please note the code above does not contain "z5messenger.StartPresence();". The reason being for that is the fact Friends List already has built-in Presence pixel therefore it can be omitted.
