Guest Handler

This handler is used in non-integrated mode with chat authorization and guest mode enabled. It checks if the login name is available to be used under guest mode.

http://yourdomain/chat71/handlers/guest.login.handler.php?userName=David

Parameters:

userName- username filled in at login screen;

The format of returned xml is almost identical to "Auto Login" and "Chat Auth" modes except for the value in < level >:

<login result="OK"> 
<userData>
  <id>2012</id>
  <name><![CDATA[David]]></name>
  <gender>0</gender>
  <location>Not specified</location>
  <age>Not specified</age>
  <photo><![CDATA[http://yourdomain/photos/nophoto_big.png]]></photo>
  <thumbnail><![CDATA[http://yourdomain/photos/nophoto_small.png]]></thumbnail>
  <details>Hello :) I am David</details>
  <level>guest</level>
</userData>
<friends> </friends>
<blocks> </blocks> 

</login >
