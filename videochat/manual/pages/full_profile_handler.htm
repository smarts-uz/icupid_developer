Full Profile handler

This handler opens new browser window when a user clicks "Full profile" link in user's profile. You may build the link the way you need it, just make sure you do not remove or modify system marker {USER_ID} which will be replaced with user ID. We have the following path for this handler in admin console:

yourdomain/profile/USER_ID/

System will open the following page in new browser tab or window:

yourdomain/profile/2010/
