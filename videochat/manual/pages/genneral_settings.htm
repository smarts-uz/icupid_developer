General settings

In settings area of the administration console you are able to tweak messenger settings:

Auto accept invitations If enabled no invitation will be shown and messenger popup will open automatically
Auto enable web cam If enabled A/V publishing will start automatically
Parse link If enabled url text in messages will be converted into clickable links.
Disconnect time Online presence disconnect time.
Auto away time If during the specified time there will be no activity in messenger window the user will automatically change status to away
Allow cam/mic access by default If enables other party receives video /audio without manual approval per user basis.
Enable away messages If enabled Messenger sends system notifications to other party once status changed to away.
